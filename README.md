# AICT v2

AICT v2 is a new Discord bot for intraday MNQ/NQ analysis using TradingView CSV uploads.

Phase 1 goals:
- auto-analyze CSV uploads in `#aict2`
- support `1 CSV` or `3 CSV` workflows
- maintain structural and session context
- consume hourly macro context from `#macro-dashboard`
- support manual `!scoredata` and `!accuracy report`

This repo is intentionally separate from AICT v1 so both bots can be compared honestly.
